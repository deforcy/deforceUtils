This is a client mod + server plugin combo of deforceUtils (my utils) that adds various helpful stuff, like:
- Noclip ![noclip icon](https://cdn.modrinth.com/data/cached_images/3da2bae69a04e0cff48bf8ec9a1df91208160e24.png)
- Infinite reach ![infinite reach icon](https://cdn.modrinth.com/data/cached_images/a9c42036d124001fdaaebbba93275fc07fc97d43.png)
- No block placement restrictions ![no block placement restriction icon](https://cdn.modrinth.com/data/cached_images/8bb1c68f02d49ad9076a2310c85c08a2cc36b0ea.png)
- No block placement update ![no block update icon](https://cdn.modrinth.com/data/cached_images/8f27775c5ff64601ba5e3a9f458ab728192db104.png)

![Player standing inside the sand block](https://cdn.modrinth.com/data/cached_images/2b74041c4a3df8f49d80344a134586de0cd7d94b.jpeg)

These features were heavily inspired by `Axiom mod` features, but implemented for 26+ versions of Minecraft and for Folia/Folia-fork server environments.

Planned new features for newer versions:
- Render entities through blocks
- Easy-to-use entity attribute editor
- Easy-to-use /give, /summon command generators
- And so on!

If you've found an issue, or got an idea, you can always report it [here](https://github.com/deforcy/deforceUtils)

_Currently close-source, but thinking about changing that in the future_
